This is a WordPress plugin for integrating WooCommerce orders with the vTiger open source CRM.

== Description ==
This plugin send a customer's information to vTiger open source CRM using the vTiger API. It checks to see if the email of the user who created the order is already an existing contact or a lead in the crm system. It will then take then either convert an existing lead from a contact, update an existing contact, or create a new contact from the order data.
